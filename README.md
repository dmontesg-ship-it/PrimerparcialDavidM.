# Programa de Facturación en C++

## Descripción
Este programa solicita el nombre del cliente, el precio del producto y la cantidad comprada.
Calcula el subtotal, el IVA (12%) y el total a pagar.

## Requisitos
- Compilador C++ 
- Consola o terminal

## Funcionalidades
- Calcula subtotal
- Calcula IVA (12%)
- Muestra total con 2 decimales
